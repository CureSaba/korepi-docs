## Monster Changer:
- When enabled, all new rendered monsters will be changed visually to the selected monster.
### Exclude Animals:
- Enable or Disable changing animals.
### Custom Monster ID:
- Use ID instead of the provided list.
### Randomize Monster:
- The change will be random.
