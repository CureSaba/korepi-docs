## Monster Changer:
- Kalo di enable, semua rendered monster akan di ganti visually ke monster yang di select.
### Exclude Animals:
- Enable atau Disable mengganti hewan.
### Custom Monster ID:
- Menggunakan ID daripada ID yang ada di list. 
### Randomize Monster:
- Perubahannya akan menjadi acak.
